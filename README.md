# Deepfake Video Detection using Deep Learning

This project detects whether a video is real or manipulated (deepfake) using deep learning techniques.  
The system extracts frames from videos, processes facial features, and predicts authenticity using a trained neural network model.

---

## Features
- Detects real vs fake videos
- Frame extraction and preprocessing
- Deep learning based classification
- Easy to test on custom videos

---

## Tech Stack
- Python
- OpenCV
- TensorFlow / PyTorch
- NumPy

---

## Project Structure
dataset/        -> sample videos or frames  
models/         -> trained weights  
src/            -> training & detection scripts  

---

## How to Run
pip install -r requirements.txt  
python predict.py
